# URL Shortener - Upskill Campus Internship

## 📌 Overview
This repository contains the final project submission for the **Free Summer Internship in Python - 2026** at Upskill Campus. The project is a functional URL Shortener application designed to take long web addresses and convert them into manageable, easily shareable links.

## 📁 Repository Structure
* **`url_shortener/`**: Contains the core Python and HTML source code for the web application.
* **`URL_Shortener_Kumar_Aman_USC_UCT.pdf`**: The final comprehensive internship project report detailing the development process, architecture, and outcomes.

## 🛠️ Tech Stack
* **Backend:** Python
* **Frontend:** HTML

## 🚀 How to Run Locally
1.  Clone this repository to your local machine:
    ```bash
    git clone [https://github.com/oblivora/upskillcampus.git](https://github.com/oblivora/upskillcampus.git)
    ```
2.  Navigate to the project directory:
    ```bash
    cd upskillcampus/url_shortener
    ```
3.  Install any necessary dependencies (if you have a `requirements.txt` file):
    ```bash
    pip install -r requirements.txt
    ```
4.  Run the application script (adjust the filename if your main entry point is different, e.g., `app.py` or `manage.py`):
    ```bash
    python main.py
    ```
